# Скрипт заполнения базы данных

Скрипт для заполнения базы данных тестовыми данными для разработки и тестирования.

## Запуск

```bash
cd backend
yarn seed
# или
npm run seed
```

## Что создается

Скрипт создает:

- **2 пользователя**:
    - Администратор: `admin@fleetmanager.ru` / `admin123`
    - Менеджер: `manager@fleetmanager.ru` / `manager123`

- **20 сотрудников** с различными должностями и отделами

- **30 автомобилей** различных марок и моделей:
    - Разные статусы (активен, в ремонте, резерв)
    - Случайный пробег
    - Некоторые с GPS-трекерами
    - Часть привязана к сотрудникам

- **25 топливных карт**:
    - Топливные и сервисные
    - Разные балансы и лимиты
    - Часть привязана к сотрудникам и автомобилям

- **500 транзакций** за период с начала 2024 года:
    - Разные типы топлива
    - Разные АЗС
    - Разные статусы (завершена, в ожидании, отменена)
    - Автоматически обновляют балансы карт и пробег автомобилей

## Важно

⚠️ **Только для разработки**: Используйте seed только в dev/test окружении. В production не запускайте скрипт без необходимости — он очищает всю базу.

⚠️ **ВНИМАНИЕ**: Скрипт полностью очищает базу данных перед заполнением!

Все существующие данные будут удалены.

Пароли по умолчанию можно переопределить через переменные окружения: `SEED_ADMIN_PASSWORD`, `SEED_MANAGER_PASSWORD`.

## Использование

После запуска скрипта вы сможете:

1. Войти в систему используя учетные данные администратора или менеджера
2. Просматривать и редактировать созданные данные
3. Тестировать функционал фильтрации и аналитики
4. Проверять работу графиков и статистики
